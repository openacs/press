sarah@arsdigita.com and stefan@arsdigita.com

The Press application allows a web site to present information and news about the site or its associated company, event, et cetera. This information is most typically presented via links to relevant off-site content.

Release Notes

The current release is 4.0b. This release depends on ACS DateTime 4.0.1b which should be available from the ACS package repository. This has been entered into the press.info file which is now available for download through cvs.arsdigita.com.

Permissions

Once you have installed and mounted the package, you need to grant the admin permission to yourself in order to access most of the pages that make this module work. Please, click here, then follow the "set permissions" link for this application to grant yourself the admin permission.
